   - `score_company(company: Company, profile: ICPPreset) -> int`:
     - Weighted scoring (0-100 scale):

       - Industry match: 35 weight (substring match against profile.industries list)
       - Geography match: 20 weight (company.country in profile.countries, include IE for Ireland per user decision)
       - Keyword match: 15 weight (profile.keywords matched against company.description, proportional)
     - Normalize to 0-100: `int(score / max_score * 100)` where max_score only counts criteria with data
     - Return 0 if no data available for any criteria
   - `batch_score_companies(companies: list[Company], profile: ICPPreset) -> list[tuple[Company, int]]`:
     - Score each company, return list of (company, score) tuples
     - Pure computation, no async needed
