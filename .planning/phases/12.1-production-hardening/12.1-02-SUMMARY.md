## User Setup Required

To enable authentication, set `APP_PASSWORD` in either:
- `.streamlit/secrets.toml`: `APP_PASSWORD = "your-password"`
- Environment variable: `APP_PASSWORD=your-password` in `.env`

If neither is set, the app warns but allows access (dev/local convenience).
